# Smart Relay Control System

A Flutter application to monitor and control multi-channel ESP32/ESP8266 relay boards via MQTT.

## Features
- **Multi-Board Support:** Add and manage multiple relay boards.
- **Real-time Control:** Toggle relays with instant feedback.
- **Timing Monitoring:** View active duration for each relay.
- **Smart Config:** Remotely configure Wi-Fi settings for new boards.
- **Localization:** Supports English and Arabic.
